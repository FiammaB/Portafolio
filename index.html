PROYECTOS UTN

Tienda de Instrumentos Musicales

Desarrollo de una **tienda online de instrumentos musicales** con funcionalidades completas de e-commerce. Permite la **gestión (CRUD)** de instrumentos, **carritos de compra y pedidos**. Incorpora un sistema de **autenticación y autorización basado en roles** (administrador, cliente), vistas dinámicas según privilegios, y **simulación de pago** con Mercado Pago. Incluye módulos de **estadísticas con gráficos** y opciones de **descarga de datos en formato Excel y PDF**. Proyecto desarrollado de forma **unipersonal** utilizando **Spring Boot (Java) para el backend, MySQL como base de datos, y React + Vite con TypeScript para el frontend**.
